# Иллюстрации к главе 2

Здесь хранятся исходники иллюстраций для главы 2 ВКР. Каждая диаграмма
подаётся в виде Mermaid-исходника (`.mmd`); для вставки в Word рендеринг
выполняется в SVG/PNG любым из способов:

1. Расширение **Markdown Preview Mermaid Support** в VS Code.
2. Онлайн-редактор `https://mermaid.live` — копируешь содержимое `.mmd`,
   экспортируешь в SVG.
3. CLI `npx -p @mermaid-js/mermaid-cli mmdc -i fig-2-5.mmd -o fig-2-5.svg`.

| Файл | Раздел | Подпись |
|---|---|---|
| `fig-2-5-formula.mmd` | 2.5 | Декомпозиция итоговой формулы ранжирования |
| `fig-2-9-canonical.mmd` | 2.4 | Канонизация типа документа: 23 наблюдаемых класса в 5 канонических |
| `fig-2-10-eval-pipeline.mmd` | 2.10 | Контур стенда экспериментальной оценки |

Прочие диаграммы (2.1 архитектура, 2.2 ER, 2.3 sequence запроса, 2.4 pipeline
загрузки) и скриншоты (2.6, 2.7, 2.8) добавляются в последующих итерациях
переработки главы.
