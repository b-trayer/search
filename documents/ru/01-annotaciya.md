# Аннотация

**Наименование темы:** проектирование и разработка поисковой подсистемы для библиотечной информационной системы с использованием технологии OpenSearch  

**Ключевые слова:** информационный поиск, OpenSearch, ранжирование, BM25, персонализация поиска, библиотечные информационные системы, CTR.

**Актуальность:** рост объема публикаций и ограниченность универсальных алгоритмов для академического контента снижают качество поиска; актуальны специализированные персонализируемые решения.

**Объект исследования:** процессы информационного поиска и ранжирования в электронных библиотечных системах.

**Цель:** поисковая подсистема на OpenSearch с настраиваемым ранжированием по профилю пользователя, типу документа и поведенческим метрикам. 

**Задачи:** провести анализ существующих методов организации поиска в библиотечных системах и выявить их ограничения; исследовать возможности OpenSearch для реализации персонализированного поиска; спроектировать архитектуру поисковой подсистемы, включая структуру индексов, обработку запросов и интеграцию с базой данных; разработать формулы ранжирования с учетом профиля пользователя, типа документа и истории взаимодействий; предложить методику оценки качества поиска, включающую стандартные IR-метрики и адаптированные показатели для профильно-поведенческих факторов; реализовать прототип поисковой подсистемы; провести пилотную оценку прототипа на стенде развертывания; разработать программу дальнейшей экспериментальной оценки качества поиска (оффлайн-протокол, проект онлайн-исследования, учет угроз валидности).

**Методы исследования:** анализ и сравнение существующих поисковых технологий, математическое моделирование функций релевантности, проектирование программной архитектуры, экспериментальные методы оценки качества поиска.

**Новизна:** разработка гибридного алгоритма ранжирования, объединяющего BM25F с компонентами персонализации (профиль пользователя, тип документа, поведенческий фактор CTR с байесовским сглаживанием), а также двухуровневого детерминированного оракула релевантности для воспроизводимой оффлайн-оценки качества поиска в библиотечных информационных системах.

**Результат:** прототип персонализированного поиска с настраиваемым ранжированием для библиотечной информационной системы.

**Область применения:** электронные библиотеки, справочные системы.
